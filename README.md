# Enoch

> *"And Enoch walked with God, and he was not, for God took him."*
> — Genesis 5:24

Enoch is a local LLM built from scratch on a curated corpus of 
scripture, mathematics, and computer science. It is designed to 
reason, retrieve, and compute — walking with its user toward understanding.

## Corpus
- Wikipedia (English)
- Religious texts: Torah, Bible, Quran, Book of Enoch, Talmud, Zohar, and more
- Mathematics: Euclid through Bourbaki
- Computer Science: K&R, SICP, Knuth, Attention Is All You Need, and more

## Architecture
GPT-2 style transformer, built in PyTorch, hardware-agnostic (CUDA / MPS / CPU)
